# Developer-Motivation-Data-Analysis-Project
🚀 Does a positive work environment significantly improve developer motivation?

This project was conducted as part of the **Theory and Practices in Statistical Modelling (TPSM)** module at SLIIT under the **Data Science specialization**.

The objective of this study is to analyze how the **work environment influences employee motivation**, using statistical modelling and data analysis techniques implemented in Python.

---

## 🎯 Problem Statement
Employee motivation plays a critical role in organizational success.  
This project investigates:

- Does the work environment significantly impact employee motivation?
- What other factors influence motivation?
- Is there a relationship between work environment and employee attrition?

---

## 🧠 Methodology

### 📌 Data Analysis Approach
We performed both **descriptive** and **inferential statistical analysis**:

- Correlation Analysis (Spearman)
- Chi-Square Test (Independence)
- Mann-Whitney U Test
- Kruskal-Wallis Test

### 📊 Tools & Technologies
- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

---

## 📈 Key Findings

- A weak negative correlation (**ρ ≈ -0.139**) indicates that poorer work environments slightly reduce employee motivation.
- Chi-Square test (**p < 0.001**) shows a significant relationship between work conditions and employee attrition.
- Work-life balance and job satisfaction have a noticeable influence on motivation.
- Motivation is not determined by a single factor but a combination of multiple variables.

---

## ⚖️ Hypothesis Testing

- **H₀ (Null Hypothesis):** Work environment has no significant impact on employee motivation  
- **H₁ (Alternative Hypothesis):** Work environment has a significant impact on employee motivation  

✅ **Conclusion:**  
The results **partially support the alternative hypothesis (H₁)**.

While statistical significance exists, the effect size is weak, indicating that **work environment alone is not a dominant factor**.  
Employee motivation is influenced by multiple factors such as:

- Work-life balance  
- Job satisfaction  
- Compensation  
- Career growth opportunities  
- Organizational culture  

---

## 🎯 Practical Implications
Organizations should adopt a **holistic strategy** to improve employee motivation rather than focusing only on the work environment.
